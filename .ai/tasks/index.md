# Task Index

## 목적
이 문서는 전역 태스크 보드예요.
어떤 태스크가 열려 있는지와 연결된 핸드오프 상태를 한눈에 관리해요.

## 태스크 목록

| task_id | 제목 | 상태 | 담당 | 핸드오프 | 마지막 업데이트 |
|---|---|---|---|---|---|

## 상태값 정의
- `TODO`: 아직 시작 전이에요.
- `IN_PROGRESS`: 현재 진행 중이에요.
- `DONE`: 수용 기준까지 완료됐어요.
- `BLOCKED`: 외부 이유나 의사결정 대기 상태예요.

## 태스크 SoT
- 태스크 기준 문서는 `.ai/tasks/<task_id>/todo.md`예요.
- 신규 태스크는 `task-init` 스킬 또는 동등한 커맨드로 생성하는 것을 권장해요.
