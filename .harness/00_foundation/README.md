# Foundation

## 목적

`00_foundation/`은 하네스 자체가 무엇을 지키는지 정의하는 최상위 규약 계층이에요.

특정 LLM에 어떻게 심는지 같은 셋업 절차는 여기 넣지 않아요.

## 포함 문서

- `philosophy.md`
- `priorities.md`
- `non-negotiables.md`
- `question-and-escalation-policy.md`
- `token-efficiency-principles.md`

## 읽기 순서

1. `philosophy.md`
2. `priorities.md`
3. `non-negotiables.md`
4. `question-and-escalation-policy.md`
5. `token-efficiency-principles.md`

## 이 폴더가 답하는 질문

| 질문 | 문서 |
|---|---|
| 하네스는 왜 존재하는가 | `philosophy.md` |
| 무엇을 우선으로 지키는가 | `priorities.md` |
| 절대 어기면 안 되는 것은 무엇인가 | `non-negotiables.md` |
| 언제 질문하고 언제 보고하는가 | `question-and-escalation-policy.md` |
| 토큰 효율은 어떻게 지키는가 | `token-efficiency-principles.md` |

## 금지

- bootstrap 절차 서술 금지
- guardian 설치 절차 서술 금지
- runtime 실행 순서 서술 금지
- repo retrofit 순서 서술 금지
