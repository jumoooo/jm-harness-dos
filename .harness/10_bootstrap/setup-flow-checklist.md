# Setup Flow Checklist

- [ ] foundation 문서를 읽었어요.
- [ ] bootstrap인지 retrofit인지 판정했어요.
- [ ] 단일 LLM 모드 가능 여부를 확인했어요.
- [ ] 복수 LLM 구성일 때 사용자 질문 필요 여부를 확인했어요.
- [ ] guardian 설치 전략을 정했어요.
- [ ] contamination 가능성을 점검했어요.
- [ ] high-impact 변경이면 사용자 승인 조건을 정리했어요.
- [ ] setup 이후 runtime 운영 규칙 연결 방식을 정의했어요.
